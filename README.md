# 🙌 프로젝트 설명
- 간단한 게시판을 구현하였습니다.
- Github Actions, Java 21 , Prometheus , Grafana , Ngrinder , Spring security , swagger doc , JPA
- Ngrinder를 통한 부하테스트와 이를 개선하는 것을 주 목적으로 하였습니다.
- API 는 기본적인 게시판 CRUD 와 로그인 회원가입이 있으며 JWT 토큰을 통한 인증 방식을 사용합니다.
